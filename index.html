Scroller Wizard

Introduction

Welcome to the ultimate media scroller setup wizard. This wizard will guide you through the process step by step. You will not be done until you make it to the final page and select Save and Finish

  1. Select scroller type
  2. Select additional options
  3. Select theme
  4. Finish/Review
Warning

Until you go through the entire process, the scroller will not be setup. If you exit the process early, you will need to start from the beginning. Once the initial setup is complete, you will be able to add/remove scenes to/from the scroller whenever you would like.

Additionally, the page must NOT be checked out by anyone. When setup is complete, the configuration will automatically check-in/publish this page.

Scroller Type

Scrollers can either be setup as simple or complex. Each has different options depending on your needs.

Features
  • Theming - Change the look and feel of the scroller from a predefined list of themes
  • Ease of use - Adding images to a picture library is extremely easy and can be done without using the scroller interface
Unavailable features
  • Audiencing - All scenes will be available to any user who can access the page *
  • Linking - You will not be able to add a link to the scene for users to click on
  • Custom Thumbnails - Allows for a custom image to be used for thumbnail navigation *
  • Scene Scheduling - You can schedule a date/time for a scene to appear *
* Some unavailable features can still be achieved by using a SharePoint view on the underlying picture library.
Features
  • Theming - Change the look and feel of the scroller from a predefined list of themes
  • Audiencing - Change which audience (group of users) can see each scene
  • Linking - You can add a link to the scene for users to click on
  • Custom Thumbnails - Allows for a custom image to be used for thumbnail navigation (Note: Not all themes allow for thumbnail navigation)
  • Scene Scheduling - You can schedule a date/time for a scene to appear
